# zy-ui

## 安裝组件zy-ui组件库
```
npm install zy-ui
yarn add zy-ui
```

### 全局导入
```
import zyUi from "zy-ui"
import "zy-ui/lib/zy-ui.css"
Vue.use(zyUi)
```

### Compiles and minifies for production
```
npm run build
```
